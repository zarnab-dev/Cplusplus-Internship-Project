# GPA Calculator (C++)

## Overview
A console-based GPA calculator that computes a student's grade point average based on subject grades and credit hours.

## Features
- Subject input handling
- Credit hour calculation
- GPA computation logic
- Structured console output

## Technical Structure
- main.cpp → Program entry
- gpa.h → Function declarations
- gpa.cpp → GPA calculation logic

## Concepts Demonstrated
- Mathematical computation
- Loop structures
- User input processing
- Modular programming design

## Purpose
To demonstrate implementation of academic grading logic using structured and maintainable C++ code.
