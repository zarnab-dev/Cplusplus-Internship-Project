# Banking Management System (C++)

## Overview
A console-based banking system designed to simulate core financial operations in a structured and modular format.

## Features
- Account creation
- Deposit functionality
- Withdrawal functionality
- Balance inquiry
- Transaction management logic

## Technical Structure
- main.cpp → Program controller
- banking.h → Function declarations
- banking.cpp → Core implementation logic

## Concepts Demonstrated
- Object-Oriented Programming
- Financial calculation logic
- Conditional and loop structures
- Modular project organization

## Purpose
To demonstrate practical application of C++ programming concepts within a financial system simulation.
